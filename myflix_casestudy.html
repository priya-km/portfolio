Overview

myFlix is a movie wiki web app that was developed using the MERN stack. It provides users with access to information about movies, directors, and movie genres. Users also have the ability to create an account, update their personal information, and create a list of favorite movies.

Purpose

myFlix was built as a personal project for my web development course at CareerFoundry to master my skills of full-stack web development with languages such as Javascript and its libraries such as React.

Objective

The goal of this project was to create the server side as well as the client side of this movie wiki app fully from scratch and host it on a live site. In the end giving me a polished project to add to my portfolio.

Approach

Server Side

The initial steps of this project involved creating a server-side RESTful API with URL endpoints and hosting movie, director, and genre information in a database.

For the server-side of the myFlix app, I developed a RESTful API using Node.js and Express. This API provides movie information in JSON format, with the data stored using MongoDB. Data retrieval and manipulation are performed using CRUD methods.

Additionally, I implemented user authentication and authorization to enhance app security. This was achieved through HTTP and JWT authentication methods. The API's functionality was thoroughly tested using Postman.





Client Side

Once the API was built, I started to build the interface in which users would see and interact with. It's a single-page, responsive application with routing, rich interactions, several interface views, and a polished user experience. All built with React.
